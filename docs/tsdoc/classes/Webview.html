bind

  • bind(name, fn): void

  • Binds a Node.js callback so that it will appear under the given name as a global JS function in webview .

    Parameters

    • name: string

      the name of the global browser's JS function

    • fn: ((w, ...args) => any)

      the callback function which receives the parameter and return the result to Webview. Any exception happened in Node.js here will reject the Promise instead of crash the program.

      Example

      bind("sumInNodeJS",(webview, arg0,arg1) => {
        return arg0+arg1;
      });

      in webview browser, you should call await sumInNodeJS(1,2) and get 3

        • (w, ...args): any

        • Parameters

          • w: Webview
          • Rest ...args: any[]

          Returns any

    Returns void

bindRaw

  • bindRaw(name, fn): void

  • Binds a native NodeJS callback so that it will appear under the given name as a global webview's JS function.

    Callback receives an Array from browser's JS. Request string is a JSON array of all the arguments passed to the JS function.

    Parameters

    • name: string

      the name of the global browser's JS function

    • fn: ((w, req) => [number, string])

      the callback function receives the request parameter in webview browser and return the response(=[isSuccess,result]), both in JSON string. If isSuccess=false, it wll reject the Promise.
